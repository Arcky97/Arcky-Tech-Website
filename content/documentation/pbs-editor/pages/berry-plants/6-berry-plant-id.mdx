## ID

### Overview

The ID is the internal identifier used by the script to refer to a Berry Plant.
Every Berry Plant must have a unique ID.
<br/>
Typically, this ID matches the Berry Plant's name (defined as an item), but written in **all capital letters**, without spaces or special characters. The editor automatically filters out invalid characters to prevent issues.
<br/>
In the PBS file, the ID is what defines the start of a new entry, and it must appear first in each Berry Plant block.
However, the editor handles this formatting for you when saving.
<br/>
---

### Default Value 

The ID is a required field and has no default value.
If the ID is not defined-or if an Berry Plant with the same ID already exists-the editor will prevent you from saving the entry.
