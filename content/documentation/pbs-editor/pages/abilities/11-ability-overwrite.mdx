## Ability Overwrite

This toggle determines how the editor handles saving the current Ability entry:

- **Off** — The editor will create a **new Ability entry**, but only if the ID does not already exist in a PBS file.
- **On** — The editor will **overwrite the existing Ability entry** that matches the current ID in the PBS file.
