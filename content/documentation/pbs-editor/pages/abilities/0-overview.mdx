## Overview

The Abilities page provides an overview of all Ability PBS files used within the project.  
It displays every default parameter defined by Essentials, as well as any additional fields introduced by plugins or custom extensions. These extra fields will automatically appear in the **Other Fields** section when relevant.
<br/>
The screenshots below illustrate the general layout and structure of the Abilities editor.  
Individual parameters and their functionality are explained in detail in the sections further down this page.  
For technical specifications or engine-level definitions, refer to the official wiki page: 
<br/> 
[Defining an Ability](https://essentialsengine.miraheze.org/wiki/Defining_an_Ability)
