## Settings

Next to the **Wiki** button in the top-right corner of the Ability Editor page, you'll find the **Settings** button.  
This opens the **PBS Editor Settings**, filtered so that only the **General Settings** and the **Ability Editor Settings** are shown.
<br/>
**The Ability Editor Settings** currently provide the following option:
<br/>
- **Ability Flags** — Allows you to add or remove ability flags so the editor can recognize and use them appropriately.
