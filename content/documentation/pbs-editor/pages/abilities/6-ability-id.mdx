## ID

### Overview

The ID is the internal identifier used by the script to refer to an Ability.  
Every Ability must have a unique ID.
<br/>
Typically, this ID matches the Ability's name, but written in **all capital letters**, without spaces or special characters.  
The editor automatically filters out invalid characters to prevent issues.
<br/>
In the PBS file, the ID is what defines the start of a new entry, and it must appear first in each Ability block.  
However, the editor handles this formatting for you when saving.
<br/>
---

### Default Value

The ID is a required field and has no default value.  
If the ID is not defined—or if an Ability with the same ID already exists—the editor will prevent you from saving the entry.
