## Import Abilities

To import one or more Abilities into the Ability Editor page, your data must be provided in a **TSV file**.
<br/>
The easiest way to create such a file is by using **Google Sheets**, which can export directly to the required TSV format.
<br/>
You can refer to the following example to see how the file should be structured: 
<br/> 
[TSV Abilities Example](https://drive.google.com/file/d/1eT6ylkMAlfq_BmMmukd9Wf5qMXLxgGX_/view?usp=sharing)
<br/>
Each Ability entry must contain the following required parameters:

- **ID**
- **Name**
- **Description**

If any of these fields are missing, the editor will skip that entry and report which required values are not present.
