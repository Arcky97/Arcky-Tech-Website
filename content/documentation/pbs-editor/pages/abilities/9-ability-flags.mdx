## Flags

### Overview

This field uses a multi-select dropdown, allowing you to choose more than one option.  
The selected flags are saved to the PBS file as a comma-separated list.

You can manage which flags appear in this dropdown through the **Ability Editor Settings**, found under the general **Settings** menu of the PBS Editor.
<br/>
---

### Default Value

Flags have no default value.
<br/>
---

### Options

The editor includes the following flag by default:

- **FasterEggHatching**

Additional flags can be added or removed through the *Ability Flags* section in the Settings.
