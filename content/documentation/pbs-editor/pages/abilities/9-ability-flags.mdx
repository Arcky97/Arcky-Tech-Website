## Flags

The Flags field is a <Gr>multi-select dropdown</Gr>, allowing you to assign multiple values to an Ability at the same time.
<br/>
All selected flags are stored in the PBS file as a <Yl>comma-separated list</Yl>, ensuring compatibility with the underlying data format.
<br/>
The available options in this dropdown can be managed through the <Gr>Ability Editor Settings</Gr>, located in the general Settings menu of the PBS Editor.
<br/>
---

### Default Value

Flags do not have a default value.
<br/>
---

### Options

By default, the editor includes the following flag:
FasterEggHatching
<br/>
Additional flags can be <Gr>added or removed</Gr> via the <Yl>Ability Flags</Yl> section in the Settings menu, allowing full customization of available flag options.
