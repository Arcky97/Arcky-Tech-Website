## Ability Selection

This dropdown menu lists all Ability entries by their ID for the currently selected file.
<br/>
In the PBS file, these IDs are written in the format `[xxx]`, but the editor displays them without the brackets for readability.  
The brackets must still be present in the actual PBS file—when saving, the editor will automatically re-add them even though they are not shown in the interface.

Ability Selection

This dropdown lists all available <Gr>Ability entries</Gr> for the currently selected file, identified by their ID.

<br/>

In the PBS file, these IDs are defined using the format <Yl>[xxx]</Yl>. For improved readability, the editor displays them <Gr>without brackets</Gr> in the interface.

<br/>

Despite this visual simplification, the brackets are still <Rd>required</Rd> in the actual PBS structure. When saving, the editor will automatically restore the correct [xxx] format, even though it is not shown within the editor itself.
