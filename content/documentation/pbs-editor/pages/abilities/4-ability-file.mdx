## File Selection

<InputSelect label="Current Ability PBS File:" width={250} initValue="abilities.txt" placeholder="Select File" options={[{ label: "abilities.txt", value: "abilities.txt" }, { label: "abilities_Gen_9_Pack.txt", value: "abilities_Gen_9_Pack.txt" }]} isClearable={false}/>

This dropdown allows you to choose between all available <Gr>Ability PBS files</Gr> detected in your project.
<br/>
The primary file is typically named `abilities.txt`, but the editor also supports additional files following the format `abilities_xxx.txt`.
<br/>
The <Yl>underscore (_)</Yl> is required for these extra files to be recognized. Any text can follow it (e.g. abilities_custom.txt), but files without this naming pattern will <Rd>not be detected</Rd> as **Ability PBS files**.
<br/>
If only a single file is available, the dropdown will be automatically <Gr>disabled</Gr>, and that file will be selected by default.
