## File Selection

This dropdown menu lists all PBS files related to Abilities.
<br/>
The main file is typically named `abilities.txt`, but additional files such as `abilities_xxx.txt` can also be used.  
The part after the underscore (`xxx`) may be any text, but the underscore itself is required — without it, the editor will not recognize the file as an Ability PBS file.
<br/>
If only a single file is available, the dropdown will be disabled, and that file will be selected automatically.
