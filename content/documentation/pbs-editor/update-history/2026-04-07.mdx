## v1.2.0: Multi Entry Tab and Import System

### New Features
- Each page has now a tab system so you can edit multiple entries at the same time.
- The import feature allows you to import and save multiple entries of a PBS type quickly to the PBS file.

### Improvements
- Priority value in the Moves Editor can now be cleared

### Bug Fixes
- Fixed a typo when creating a new item, the field was named `portionPlural` instead of `portionNamePlural`.
- Fixed a type when creating a new Map Metadata, the field was named `wildBattleBgm` instead of `wildBattleBGM`.
