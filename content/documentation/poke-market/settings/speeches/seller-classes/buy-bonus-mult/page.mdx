<Header title="BuyBonusMult">
  How to use and define BuyBonusMult speeches.
</Header>

The `BuyBonusMult` determines what the NPC will say when the player receives 1 or multiple bonus Items.
<br/>
See the [BonusItems](../../../settings/bonus-items) page for more information.

<PMSpeeches speech="BuyBonusMult"/>

To define the `BuyBonusMult`, use this structure:

<br/>
```ruby
BuyBonusMult: [
  "And have {1} on the house!"
]
```
