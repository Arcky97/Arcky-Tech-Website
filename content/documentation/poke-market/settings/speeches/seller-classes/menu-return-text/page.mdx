<header>
  # MenuReturnText
  <br/>
  How to use and define MenuReturnText speeches.

  ---
</header>

The MenuReturnText is what the NPC will say when a chosen Item can't be sold.
<br/>
It is possible to include more than 1 speech if you want to but you can't add a Time or Day.
<br/>
Defining `MenuReturnText` looks like this:
<br/>
```ruby
MenuReturnText: [
  "Is there anything else I can do for you?", 
  "What else could I mean for you today?"
]
```
