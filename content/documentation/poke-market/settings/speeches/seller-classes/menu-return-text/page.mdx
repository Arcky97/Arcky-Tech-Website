<Header title="MenuReturnText">
  How to use and define MenuReturnText speeches.
</Header>

The `MenuReturnText` determines what the NPC will say when the player returns back to the options menu (the same menu given with the `IntroText`).

<PMSpeeches speech="MenuReturnText"/>

To define the `MenuReturnText`, use this structure:

```ruby
MenuReturnText: [
  "Is there anything else I can do for you?", 
  "What else could I mean for you today?"
]
```
