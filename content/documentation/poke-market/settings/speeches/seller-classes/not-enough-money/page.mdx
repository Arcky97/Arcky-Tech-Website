<header>
  # NotEnoughMoney
  <br/>
  How to use and define NotEnoughMoney speeches.

  ---
</header>

The NotEnoughMoney is what the NPC will say when you don't have enough money to buy an Item.
<br/>
It is possible to include more than 1 speech if you want to but you can't add a Time or Day.
<br/>
Defining `NotEnoughMoney` looks like this:
<br/>
```ruby
NotEnoughMoney: ["You don't have enough money."]
```
