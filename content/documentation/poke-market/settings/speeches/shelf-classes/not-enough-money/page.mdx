<header>
  # NotEnoughMoney
  <br/>
  How to use and define NotEnoughMoney speeches.

  ---
</header>

The NotEnoughMoney is what the shelf will say when selecting an item and you don't have enough money to add it to your basket.
<br/>
This only happens when you haven't added this item yet.
<br/>
It is possible to include more than 1 speech if you want to.
<br/>
Defining `NotEnoughMoney` looks like this:
<br/>
```ruby
NotEnoughMoney: [
  "You don't have enough {1} to add any {2} to your basket."
]
```
- \{1\} = the currency set for this shelf (for ex. "Money")

- \{2\} = chosen Item's name (plural) (for ex. "Super Potions").
