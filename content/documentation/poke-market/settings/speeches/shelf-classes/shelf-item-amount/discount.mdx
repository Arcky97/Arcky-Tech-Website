## ShelfItemAmountDiscount

The ShelfItemAmountDiscount is what the shelf will say when selecting an item and there's a discount defined for the shelf event.
<br/>
It is possible to include more than 1 speech if you want to.
<br/>
Defining `ShelfItemAmountDiscount` looks like this:
<br/>
```ruby
ShelfItemAmountDiscount: [
  "There's a discount {1} on {2}, how many would you like?"
]
```
- \{1\} = "of" + percentage of discount (for ex.11%).

- \{2\} = chosen item's name (plural) (for ex. "Potions").

<ImageWithCaption src="/images/documentation/poke-market/ShelfItemAmountDiscount 1.png" caption="ShelfItemAmountDiscount 1"/>
You can also only use \{2\}:
<br/>
```ruby
ShelfItemAmountDiscount: [
  "There's a discount on {2}, how many would you like?"
]
```
