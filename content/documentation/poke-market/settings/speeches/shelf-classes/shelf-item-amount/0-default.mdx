## ShelfItemAmount

The `ShelfItemAmount` determines what the player will be prompted with when the selected Item wasn't added to the Basket yet.

<SHSpeeches speech="ShelfItemAmount"/>

To define the `ShelfItemAmount`, use this structure:
<br/>
```ruby
ShelfAmountItem: [
  "How many {1} would you like to add to your basket?"
]
```
