<Header title="IntroSpecies">
  How to use and define IntroSpecies speeches.
</Header>

The `IntroSpecies` determines what the NPC says when you first talk to them (i.e., at the start of the interaction).

<SPSpeeches speech="IntroSpecies"/>

To define the `IntroSpecies`, use this structure:
<br/>
```ruby 
IntroSpecies: [
  "Hey you! Want to see my exclusive offer?", 
  "Hello, are you interested in my exclusive offer perhaps?"
]
```
