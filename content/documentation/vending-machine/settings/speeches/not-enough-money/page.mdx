<Header title="NotEnoughMoney">
  How to use and define various NotEnoughMoney speeches.
</Header>

The `NotEnoughMoney` determines what the Vending Machine will say when you don't have enough money.
<br/>
As shown in the example below, you can include multiple variations. The script will randomly choose one line from the array each time the player interacts with the Vending Machine Event.
<br/>
To define the `NotEnoughMoney`, use this structure:

```ruby 
NotEnoughMoney: [
  "Sorry, you seem to have not enough money."
]
```
