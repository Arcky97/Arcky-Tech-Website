## Installing the Plugin for the first time
The installation process for this plugin is mostly the same for v20.1 and v21.1 but there are a few differences in where to put your own files. Please do not follow the installation process if you've installed this plugin before, the process for updating is [explained on this page here](/documentation/region-map/updating).
