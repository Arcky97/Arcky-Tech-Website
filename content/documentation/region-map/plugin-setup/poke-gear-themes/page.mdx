## Overview
In first case, I'm talking about [Lin's Poke Gear Themes plugin](https://eeveeexpo.com/resources/1321/).
<br/>
There's no additional Setup required to make this plugin work together with the Region Map plugin. 
<br/>All required Graphics are included with this plugin.
<br/>
The Region Map Plugin automatically detects when this plugin is installed and will use the Poke Gear theme accordingly.
