## v3.3.0: New Pokedex Features

<Callout type="attention" blend="light">
  This plugin (update) requires [Arcky's Utilities](https://drive.google.com/file/d/19xrRavd1CpbC0p1UQtOUZzuEeHL_WHB5/view?usp=sharing) v1.0.9 or higher to function correctly. Once you've install this, start a new save to avoid malfunction.
</Callout>

### New Features:

- It is now possible to change the Current Region in the Pokedex Area Section.
- You can now filter the Location Areas you can find a species in. Filtering is possible on both Location and Encounter Type.

### New Settings:

- Added a new Setting `ToggleRegionSwitchButton` which allows you to choose which button needs to be pressed to change the Region in the Pokedex' Area Page.
- Added a new Setting `ToggleLocFilterButton` which allows you to choose which button needs to be pressed to trigger the Location Filter in the Pokedex' Area Page.
- Added a new Setting `ToggleEncTypeFilterButton` which allows you to choose which button needs to be pressed to trigger the Encounter Type Filter in the Pokedex' Area Page.
- Added a new Setting `FilterMenuType` which allows you to change the behavior of the filters. See [Settings](./settings/pokedex#filtermenutype)
