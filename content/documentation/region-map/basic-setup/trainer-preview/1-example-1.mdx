## Example 1

In the first example, a Trainer Rematch is registered by using the Self Switch B of the Event. The Trainer Icon will be displayed on the Region Map as soon as the Self Switch is set to `ON`.

```ruby
TrainerRematchesConfig = {
  "B" => {
      name: "Liam",
      type: "CAMPER",
      event: 4,
      map: 10
    },
  }
```
