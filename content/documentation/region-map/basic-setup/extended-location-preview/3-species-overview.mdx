## Species Overview

This will show the cursor on the first revealed Species and will show you the following information:
<br/>
### Page 1
- Species Type: shows what Type the current selected Species is.
- Catch Rate: shows the Catch Rate of the current selected Species.
- Encounter Rate: shows all Encounter Rates (if more than 1) of the current selected Species.
<ImageWithCaption src="/images/documentation/region-map/Extended Preview 12.png" caption="Species Page 1 Route 1" align="left" size="medium"/>
<ImageWithCaption src="/images/documentation/region-map/Extended Preview 13.png" caption="Species Page 1 Route 2" align="left" size="medium"/>

### Page 2
- Counters
  - Seen Counter: shows how many you've seen on this Game Map. <br/>When you've seen the Species on 1 other or more Game Maps, it'll also give you the total number.
  - Caught Counter: shows how many you've caught on this Game Map. <br/>When you've caught the Species on 1 other or more Game Maps, it'll also give you the total number.
  - Defeated Counter: shows how many you've defated on this Game Map. <br/>When you've defeated the Species on 1 other or more Game Maps, it'll also give you the total number.

<ImageWithCaption src="/images/documentation/region-map/Extended Preview 14.png" caption="Species Page 2 Route 1" align="left" size="medium"/>
Rattata was seen on Route 2, so it'll give you the total number of seen Rattata on all Routes. The total number will not be shown if it's the same or 0.
