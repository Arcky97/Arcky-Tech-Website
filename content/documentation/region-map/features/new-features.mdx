## New Features (Added up to v3.3.0)
<Callout type="random" title="District Names">
  Setup a different Region Name for certain parts on the Region Map.
</Callout>
<Callout type="random" title='Replace Unvisited Locations with "???"'>
  Setup a placeholder text for unvisited maps so the name of unvisited maps are not shown yet on the Region Map.
</Callout>
<Callout type="random" title="Multiple Mode Option Menu">
  Gives a menu with all available Modes instead of having to press the "Change Mode" button several times to get to the wanted Mode.
</Callout>
<Callout type="random" title="Custom Music">
  Change the music that plays while the Region Map is active.
</Callout>
<Callout type="random" title="Bigger Screen Size Support">
  Gives support for game screen sizes taht are different than the default size of 512x384.
</Callout>
<Callout type="random" title="Detailed Location Preview">
  Setup and show detailed information about location on your Region Map.
</Callout>
<Callout type="random" title="Region Map Switch with menu">
  Switch between different Regions in your game.
</Callout>
<Callout type="random" title="Change the UI by Region">
  Setup different UI Graphics for each Region in your game.
</Callout>
<Callout type="random" title="Game Translation Support">
  Supports game translation if any has been applied.
</Callout>
<Callout type="random" title="Roaming Icons">
  Shows Roaming Species on the Region Map (Roaming Mode) if any are active.
</Callout>
<Callout type="random" title="Location Searcher with filter option">
  Gives a list with available locations and allows you to search through them as well.
</Callout>
<Callout type="random" title="Fake Locations">
  Setup multiple Town Map positions for a single Game Map.
</Callout>
<Callout type="random" title="Progress Counter">
  Shows the progress of Maps visited, Trainers defeated, Items found and Species seen and caught. 
</Callout>
<Callout type="random" title="Mouse Support">
  Allows you to select a location, move the map and interact with a location (like open a Location, Quest, Berry or Trainer Preview or select that location to Fly to).
</Callout>
<Callout type="random" title="Region Map Time Dependable">
  Setup a Region Map Graphic dependable on the current time in the Game.
</Callout>
<Callout type="random" title="Extended Location Preview">
  Shows more information of a location if a Location Preview has been setup for that location (and enabled globally).
</Callout>
<Callout type="random" title="Region Map Zooming">
  You can zoom in/out on the Region Map.
</Callout>
<Callout type="random" title="Preview Animations">
  Preview animations are more smooth now rather than being shown instantly.
</Callout>
<Callout type="random" title="Region Map Connecting">
  Setup several regions and connect them together to make it look like they form a bigger Map together.
</Callout>
<Callout type="random" title="Trainer Mode">
  Tracks and shows Trainers that are ready for rematches. Supports different setup methods. 
</Callout>
<Callout type="random" title="Pokedex Region Changing">
  Change the Current Region in the Pokedex Area Page.
</Callout>
<Callout type="random" title="Pokedex Area Page Filters">
  Filter the Location Areas you can find a species in. Filtering is possible on both Location and Encounter Type.
</Callout>
