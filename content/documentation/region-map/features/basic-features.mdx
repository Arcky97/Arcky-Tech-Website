## Basic Features 
<Callout type="random" title="Bigger Map Support">
  Offers support for bigger Region Map Sizes that exceed the default size of 480x320.
</Callout>
<Callout type="random" title="Location Highlighting">
  Highlights the current location the cursor is on.
</Callout>
<Callout type="random" title="Unvisited Location Marking">
  Puts an unvisited Graphic on the locations that haven't been visited yet. (Only applies to locations with a fly spot defined.)
</Callout>
<Callout type="random" title="Pokedex Area Map Scrolling" >
  Just like the Region Map, the Pokedex Area Page allows you to scroll the map as well (if applyable).
</Callout>
<Callout type="random" title="Smart Option Selection">
  While on the Region Map, the script will remember the last chosen option in the choice menus.
</Callout>
