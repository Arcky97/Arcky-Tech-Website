## Features that require other plugins:
<Callout type="random" title="Quest Icons and Quest Preview">
  - Requires: [Modern Quest System](https://eeveeexpo.com/resources/709/)

  - These Icons will only show on the Quest Mode, if there are any active quests and if those or any quests have a map position defined.
</Callout>
<Callout type="random" title="Poke Gear Themes">
  - Requires: [Lin's Poke Gear Themes](https://eeveeexpo.com/resources/1321/)

  - All Graphics required by the Region Map Script are included with this plugin in their respective Theme Folder.
</Callout>
<Callout type="random" title="Weather Preview">
  - Requires: [Lin's Weather System](https://eeveeexpo.com/resources/1411/)
  - Shows in a small Preview Box the Weather of the Current Location.
</Callout>
<Callout type="random" title="Set the Controls Screen">
  - Requires: [FL's Set the Control Screen](https://eeveeexpo.com/resources/1038/)
  - The Region Map Script will convert the Buttons you've set for each Action to the actual keyboard key that was set by this Plugin. For ex.: If you set the "space" key for "Action", in the Region Map, it'll then say "Space: Show Info".
</Callout>
<Callout type="random" title="Berry Icons and Preview">
  - Requires: [Wrigty12's Berry Planting Improvements](https://eeveeexpo.com/resources/1410/
)
  - Shows a preview of Berries you have planted across location in your Region. It'll give basic information like berry name and their growth state. This includes Icons for each Growth State as well.
</Callout>
