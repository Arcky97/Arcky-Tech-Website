## Extended Preview Percentage

Use this Setting to enable or disable the Progress Counter Percentage shown on the Extended Location Preview, next to the Location Name.

<SetVer
  v211="DisableExtendedPreviewPercentage = false"
  v201="DISABLE_EXTENDED_PREVIEW_PERCENTAGE = false"
/>

### Possible Values 

- `true` = Progress Percentage is **<Rd>not shown</Rd>**.
- `false` = Progress Percentage is **<Gn>shown</Gn>**.
