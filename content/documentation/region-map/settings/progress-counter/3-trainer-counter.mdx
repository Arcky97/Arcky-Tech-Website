## Trainer Tracking

Use this Setting to enable or disable the Progress Counter Tracking of defeated Trainers.

<SetVer
  v211="ProgressCountTrainer = true"
  v201="PROGRESS_COUNT_TRAINER = true"
/>

### Possible Values 

- `true` = Defeated Trainers are **<Gn>tracked</Gn>**.
- `false` = Defeated Trainers are **<Rd>not tracked</Rd>**.
