## Location Visit Tracking

Use this Setting to enable or disable the Progress Counter Tracking of Visited Locations.

<SetVer
  v211="ProgressCountVisitedLocations = true"
  v201="PROGRESS_COUNT_VISITED_LOCATIONS = true"
/>

### Possible Values 

- `true` = Visited Locations are **<Gn>tracked</Gn>**.
- `false` = Visited Locations are **<Rd>not tracked</Rd>**.
