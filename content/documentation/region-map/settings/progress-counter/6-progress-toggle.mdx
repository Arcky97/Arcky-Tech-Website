## Progress Counter Percentage

Use this Setting to enable or disable the Progress Counter Percentage being shown on the Region Map UI, next to the Region Name.

```ruby 
DisableProgressCounterPercentage = false 
```

### Possible Values 

- `true` = Progress Percentage is **<Rd>not shown</Rd>**.
- `false` = Progress Percentage is **<Gn>shown</Gn>**
