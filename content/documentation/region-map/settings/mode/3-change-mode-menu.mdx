## ChangeModeMenu

Use this Setting wheter to enable or disable the usage of a Choice Menu when 3 or more Modes are available to change to on the Region Map.

```ruby
ChangeModeMenu = true
```

### Possible Values 

- `true` = A menu will **<Gn>be shown</Gn>** for quicker mode changing.
- `false` = A menu will **<Rd>not be shown</Rd>**, mode changes by each [ChangeModeButton](#changemodebutton) usage.
