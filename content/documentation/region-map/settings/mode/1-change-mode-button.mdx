## ChangeModeButton

Use this Setting to set which button needs to be pressed to change the Mode on the Region Map.

```ruby
ChangeModeButton = Input::ACTION
```
### Possible Values

- `Input::ACTION` = The Game Button.

<Callout type="attention">
  If you set this to the same button as for QuickFly or any kind Preview Viewing, then you won't be able to change Modes anymore.
</Callout>
