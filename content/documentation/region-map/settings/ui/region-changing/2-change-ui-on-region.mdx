## ChangeUIOnRegion

Use this Setting to enable or disable the Region Map UI changing on Region change.
<br/>Region Specific UI's should go in the `/UI/Region(ID)`. If a specific Region UI is not found, the one in `/UI/Default` or `/UI/Special` will be used instead.

```ruby
ChangeUIOnRegion = false
```

### Possible Values

- `true`= The UI **<Gn>will change</Gn>** on Region Change.
- `false` = The UI **<Rd>will not change</Rd>** on Region Change.
