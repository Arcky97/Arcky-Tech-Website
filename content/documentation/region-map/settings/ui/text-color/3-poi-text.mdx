## PoiTextColor

Use these Settings to change the main and shadow color of the POI (Point of Interest) Name Text on the Region Map.

<SetVer
  v201={`POI_TEXT_MAIN = Color.new(248, 248, 248)
POI_TEXT_SHADOW = Color.new(0, 0, 0)`}
  v211={`PoiTextMain = Color.new(248, 248, 248)
PoiTextShadow = Color.new(0, 0, 0)`}
/>

### Possible Values

The 3 arguments passed to `Color.new()` must have a value between `0` and `255` and represent Red, Green and Blue respectivly.
