## RegionTextColor

Use these Settings to change the main and shadow color of the Region/District Name Text on the Region Map.

```ruby
RegionTextMain = Color.new(248, 248, 248)
RegionTextShadow = Color.new(0, 0, 0)
```

### Possible Values 

The 3 arguments passed to `Color.new()` must have a value between `0` and `255` and represent Red, Green and Blue respectivly.
