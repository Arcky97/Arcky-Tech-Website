## CenterCursorByDefault

Use this Setting to enable or disable the Cursor being centered on the Region Map when there's no `mapPosition` defined for the Game Map the Region Map was opened from.

<SetVer
  v201="CENTER_CURSOR_BY_DEFAULT = true"
  v211="CenterCursorByDefault = true"
/>

### Possible Values

- `true` = The cursor will **<Gn>be centered</Gn>**.
- `false` = The cursor will **<Rd>not be centered</Rd>** and placed at the top left.
