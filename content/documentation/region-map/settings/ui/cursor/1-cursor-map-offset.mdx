## CursorMapOffset

Use this Setting to enable or disable a 1 tile offset on the Region Map.

```ruby
CursorMapOffset = true
```

### Possible Values

- `true` = There will be **<Gn> a 1 tile offset</Gn>** towards the edges of the Region Map.
- `false` = There will be **<Rd>no offset</Rd>** towards the edges of the Region Map.
