## ToggleZoomButton

Use this Setting to set which button needs to be pressed to toggle the Zoom Mode on the Region Map.

```ruby 
ToggleZoomButton = Input::CTRL 
```

### Possible Values 

- `Input::CTRL` = The Game Button.

<Callout type="hint">
  Be careful with which button you set this to as it can conflict with other button feature triggers.
</Callout>
