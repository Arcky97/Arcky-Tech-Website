## UseMouseOnRegionMap

Use this Setting to enable or disable the support of the Mouse on the Region Map.

<SetVer
  v201="USE_MOUSE_ON_REGION_MAP = true"
  v211="UseMouseOnRegionMap = true"
/>

### Possible Values
- `true` = Mouse Support is **<Gn>enabled</Gn>**.
- `false` = Mouse Support is **<Rd>disabled</Rd>**.
