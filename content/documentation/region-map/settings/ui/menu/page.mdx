## Function of the Setting

The **Menu Settings Section** includes a single setting. It allows you to change the max number of items are shown at once in a choice menu. This might prevent the Region Map to be half covered with the choices to choose from.

<SetVer
  v201="MAX_OPTIONS_CHOICE_MENU = 4"
  v211="MaxOptionsChoiceMenu = 4"
/>

--- 

## Possible values

- `4` = The max items shown at once in a choice menu.
