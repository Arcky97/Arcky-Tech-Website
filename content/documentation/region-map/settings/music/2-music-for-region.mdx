## MusicForRegion

Use this setting to add BGM (Background Music) by Region.

```ruby
MusicForRegion = [
  [0, "Radio - Oak", 90, 100],
  [1, "Radio - March"]
]
```

### Possible Values

Each Item you add here must contain the following:
- `0` = Region Map ID.
- `Radio - Oak` = The BGM name.
- `90` = The Volume of the Music.
- `100` = The Pitch of the Music.
