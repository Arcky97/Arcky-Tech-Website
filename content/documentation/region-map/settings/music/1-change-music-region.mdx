## ChangeMusicInRegionMap

Use this Setting to enable or disable the Music changing when the Region Map is being opened.

<SetVer
  v201="CHANGE_MUSIC_IN_REGION_MAP = true"
  v211="ChangeMusicInRegionMap = true"
/>

### Possible Values 

- `true` = The BGM (Background Music) **<Gn>will change</Gn>**.
- `false` = The BGM (Background Music) **<Rd>will not change</Rd>**.
