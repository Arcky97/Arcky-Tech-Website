## QuickSearchButton

Use this Setting to set which button needs to be pressed to activate the Quick Search function while using the Location Search Feature on the Region Map.

```ruby
QuickSearchButton = Input::ACTION
```

### Possible Values 

- `Input::ACTION` = the Game Button.

<Callout type="hint">
  You may set this to the same button as the `LocationSearchButton` but **<Rd>do not set this to USE or BACK</Rd>**.
</Callout>
