## IncludeUnvisitedMaps

Use this Setting to whether or not include Unvisited Locations in the search result.

```ruby 
IncludeUnvisitedMaps = false
```

### Possible Values

- `true` = Unvisited Locations will **<Gn>be excluded</Gn>** in the search result list.
- `false` = Unvisited Locations will **<Rd>be included</Rd>** in the search result list.
