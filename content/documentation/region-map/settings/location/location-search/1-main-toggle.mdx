## CanLocationSearch

Use this Setting to enable or disable the Location Search Feature on the Region Map.

<SetVer
  v201="CAN_LOCATION_SEARCH = true"
  v211="CanLocationSearch = true"
/>

### Possible Values 

- `true` = Location Search can **<Gn>be used</Gn>**.
- `false` = Location Search can **<Rd>not be used</Rd>**.
- `Switch ID` = Location Search can **<Or>Only be used</Or>** when the Game Switch is `ON`.
