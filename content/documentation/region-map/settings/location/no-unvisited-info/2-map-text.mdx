## Locations Replace Text

This Setting may be used to set the replace name of Unvisited Locations

```ruby 
UnvisitedMapText = "???"
```

### Possible Values

The value must be a string between `" "` or `' '`. 
<Callout type="hint">
  Best practice is to set it to something short and clear from the start like `"Unkown"` or `"???"`
</Callout>
