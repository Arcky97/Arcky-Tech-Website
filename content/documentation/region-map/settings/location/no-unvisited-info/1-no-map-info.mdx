## Main Toggle

Use this Setting to enable or disable the revealing of the names of unvisited Locations on the Region Map.

<SetVer
  v211="NoUnvistedMapInfo = false"
  v201="NO_UNVISITED_MAP_INFO = false"
/>

### Possible Values

- `true` = Unvisited Location names will **<Gn>not be revealed</Gn>**.
- `false` = Unvisited Location names will **<Rd>be revealed</Rd>**.
