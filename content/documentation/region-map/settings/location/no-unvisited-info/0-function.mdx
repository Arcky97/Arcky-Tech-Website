## Function of the Setting

The **No Unvisited Location Info Settings Section** inclused various Settings to whether or not reveal the names of Unvisited Locations on the Region Map.
<br/>It also allows you to:
- Set the replace text for unvisited
  - map locations 
  - point of interest locations

And optionally, link Game Map IDs to a point of interest on the Region Map for more accurate revealing once visited.
