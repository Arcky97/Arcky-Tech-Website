## Function of the Setting 

The **Location Highlight Setting Section** includes a single Setting. It allows you to change the opacity of the Highlight Graphic, placed on top of the current location on the Region Map (only defined locations in the town_map.txt PBS file).

```ruby 
HighlightOpacity = 90
```

---

## Possible Values 

- `90` = The Highlight Graphics will have an opacity of 90%.

<Callout type="hint">
  When a value of, for ex., `93` is given, the value will be rounded up to `95`. For a value of `81` will be rounded to `80`.
</Callout>
