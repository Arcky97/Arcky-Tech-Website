## UseRegionDistrictsNames

Use this Setting to enable or disable the Region Districts Feature on the Region Map.

<SetVer
  v201="USE_REGION_DISTRICTS_NAMES = true"
  v211="UseRegionDistrictsNames = true"
/>

### Possible Values

- `true` = Region Districts will **<Gn>be used</Gn>**.
- `false` = Region Districts will **<Rd>not be used</Rd>**.
