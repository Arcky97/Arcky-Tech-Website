## AllowFlyToOtherRegions

Use this Setting to whether or not allow flying from 1 Region to another.

```ruby 
AllowFlyToOtherRegions = true
```

### Possible Values

- `true` = The player is **<Gn>allowed to fly</Gn>** to another Region.
- `false` = The player is **<Rd>not allowed to fly</Rd>** to another Region.
