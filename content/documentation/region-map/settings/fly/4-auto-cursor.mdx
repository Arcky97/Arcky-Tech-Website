## AutoCursorMovement

Use this Setting to whether or not make the cursor move to the selected location on the Quick Fly Menu (before confirming the selected location).

<SetVer
  v201="AUTO_CURSOR_MOVEMENT = true"
  v211="AutoCursorMovement = true"
/>

### Possible Values 

- `true` = The Cursor will move **<Gn>on location selecting</Gn>**.
- `false` = The Cursor will move **<Rd>on location confirming</Rd>**.
