## FlyToRegions

Use this Setting to set from which Regions you can Fly to another Region.

<SetVer
  v201={`FLY_TO_REGIONS = {
    :Essen => [1, 3],
    :Tiall => [0, 3]
  }`}
  v211={`FlyToRegions = {
    :Essen => [1, 3],
    :Tiall => [0, 3]
  }`}
/>

### Possible Values

- `:Essen` = Region name
  - `[1, 3]` = The Region IDs you can fly to from the `Essen` Region.
