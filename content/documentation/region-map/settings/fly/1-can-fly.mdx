## CanFlyFromTownMap

Use this Setting to enable or disable using Fly on the Region Map (Pokegear or Item only). This Setting does not effect using the Field Move Fly.

<SetVer
  v201="CAN_FLY_FROM_TOWN_MAP = true"
  v211="CanFlyFromTownMap = true"
/>

### Possible Values

- `true` = Player **<Gn>can use</Gn>** Fly on the Region Map.
- `false` = Player **<Rd>cannot use</Rd>** Fly on the Region Map. (Only through the Field Move Fly.)
