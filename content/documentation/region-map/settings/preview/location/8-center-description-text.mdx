## CenterDescriptionText

Use this Setting to whether or not center the Location Preview Description Text on the Region Map. This is mostly the case in combination with an Location Preview Icon.

<SetVer
  v201="CENTER_DESCRIPTION_TEXT = true"
  v211="CenterDescriptionText = true"
/>

### Possible Values

- `true` = The Description Text will **<Gn>be centered vertically</Gn>** when the total line height is smaller than the Icon's height.
- `false` = The Description Text will **<Rd>not be centered vertically</Rd>**.
