## CenterIcon

Use this Setting to whether or not center the Location Preview Icon on the Region Map. This is mostly the case in combination with the Location Preview Description Text. This Setting is similar to the [CenterDescriptionText](#centerdescriptiontext) Setting.

<SetVer
  v201="CENTER_ICON  = true"
  v211="CenterIcon = true"
/>

### Possible Values 

- `true` = The Icon will **<Gn>be centered vertically </Gn>** when the Icon height is less than the total line height of the Description's Text.
- `false` = The Icon will **<Rd>not be centered vertically</Rd>**.
