## DirectionHeightSpacing

Use this Setting to change the vertical space between the Location Preview Description and the Location Preview Direction Text.

<SetVer
  v201="DIRECTION_HEIGHT_SPACING = 16"
  v211="DirectionHeightSpacing = 16"
/>

### Possible Values

- Any value above 0.

<Callout type="hint">
  Just keep in mind that changing this value to a higher value might require the Location Preview Box Graphics to be adjusted so everything still fits in.
</Callout>
