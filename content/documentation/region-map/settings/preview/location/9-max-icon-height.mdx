## MaxIconHeight

Use this Setting to change the max height a location Icon can take on the Location Preview. 

<SetVer
  v201="MAX_ICON_HEIGHT = 96"
  v211="MaxIconHeight = 96"
/>

### Possible Values

- Any number above 0 but the value should be equal to [PreviewLineHeight](general#previewlineheight) * [MaxDescriptionLines](#maxdescriptionlines).
