## LocationDirectionSpaces

Use this Setting to adjust the space between each individual item in the Direction Item on the Location Preview.

```ruby
LocationDirectionSpaces = 3
```

<SetVer
  v201="LOCATION_DIRECTION_SPACES = 3"
  v211="LocationDirectionSpaces = 3"
/>

### Possible Values 

- `3` = 3 spaces will be added between each Direction Item in the Location Direction Text.
