## UnvisitedMapInfoText

Use this Setting to set the default Text to display in the Location Preview for Unvisited Locations on the Region Map. This Setting has only effect when [CanViewInfoUnvisitedMaps](#canviewinfounvisitedmaps) is enabled.

```ruby
UnvisitedMapInfoText = _INTL("No information Available")
```

### Possible Values 

- Any Default Text for Unvisited Locations.
