## ShowLocationButton

Use this Setting to set which button needs to be pressed to show the Location Preview for a Location on the Region Map.

<SetVer
  v201="SHOW_LOCATION_BUTTON = Input::USE"
  v211="ShowLocationButton = Input::USE"
/>

### Possible Values 

- `Input::USE` = The Game Button.

<Callout type="hint">
  Be careful with which button you set this to as it can conflict with another button feature trigger.
</Callout>
