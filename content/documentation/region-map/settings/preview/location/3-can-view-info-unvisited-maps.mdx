## CanViewInfoUnvisitedMaps

Use this Setting whether or not you want the Location Preview to be available for Locations that have not been visited yet.

<SetVer
  v201="CAN_VIEW_INFO_UNVISITED_MAPS = true"
  v211="CanViewInfoUnvisitedMaps = true"
/>

### Possible Values 

- `true` = Location Info of unvisited maps **<Gn>can be viewed</Gn>**.
- `false` = Location info of unvisited maps **<Rd>cannot be viewed</Rd>**.
