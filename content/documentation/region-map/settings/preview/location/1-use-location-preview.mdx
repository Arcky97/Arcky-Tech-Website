## UseLocationPreview

Use this Setting to enable or disable the Location Preview Feature on the Region Map. This does require aditional setup in the `001_RegionMap_LocationInfo.rb`. See [Location Preview Setup](../../basic-setup/location-preview) for more information about the Setup.

```ruby
UseLocationPreview = true
```

### Possible Values

- `true` = The Location Preview **<Gn>can be used</Gn>**.
- `false` = The Location Preview **<Rd>cannot be used</Rd>**.
