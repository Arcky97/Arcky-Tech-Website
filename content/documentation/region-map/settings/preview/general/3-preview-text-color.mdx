## PreviewTextColor

Use these Settings to change the main and shadow color of the Quest and Berry Preview Name Text on the Region Map.

```ruby 
PreviewTextMain = Color.new(248, 248, 248)
PreviewTextShadow = Color.new(0, 0, 0)
```

### Possible Values

The 3 arguments passed to `Color.new()` must have a value between `0` and `255` and represent Red, Green and Blue respectivly.
