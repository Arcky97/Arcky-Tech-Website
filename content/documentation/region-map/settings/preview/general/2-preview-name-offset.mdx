## PreviewNameOffset

Use these Settings to add a small Offset to the X and/or Y Position of the Quest and Berry Preview Name Text on the Region Map.

<SetVer
  v201={`QUEST_AND_BERRY_NAME_OFFSET_X = 0
QUEST_AND_BERRY_NAME_OFFSET_Y = 0`}
  v211={`PreviewNameOffsetX = 0
PreviewNameOffsetY = 0`}
/>

### Possible Values

The offset value may as well be nagative as positive depending on your needs. **Only round numbers are allowed**.
