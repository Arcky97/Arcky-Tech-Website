## SelectSpeciesButton

Use this Setting to set which button needs to be pressed to select a species on the Sub Page of the Extended Location Preview and view additional information about this species.

```ruby 
SelectSpeciesButton = Input::USE
```

### Possible Values

- `Input::USE` = The Game Button.

<Callout type="hint">
  Once again, there's no problem with setting this to the same button as for [ShowExtendedButton](#showextendedbutton) and/or [ShowExtendedSubButton](showextendedsubbutton) as it's another follow up section of the Extended Location Preview.
</Callout>
