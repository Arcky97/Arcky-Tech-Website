## WeatherOnLocationPreviewActive

Use this Setting to enable or disable the Weather Preview being shown together when the Location Preview is active.

<SetVer
  v201="WEATHER_ON_LOCATION_PREVIEW_ACTIVE = true"
  v211="WeatherOnLocationPreviewActive = true"
/>

### Possible Values

- `true` = The Weather Preview is **<Gn>shown when the Location Preview is active</Gn>**.
- `false` = The Weather Preview is **<Rd> always shown on Normal Mode</Rd>**.
