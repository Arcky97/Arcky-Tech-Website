## WeatherOnModes

Use this Setting to set which modes can show the Weather Preview.

```ruby
WeatherOnModes = [0, 1, 2]
```

### Possible Values

Each item in the Array is the ID of the Region Map Modes.
- 0 = Normal Map 
- 1 = Fly Map 
- 2 = Quest Map 
- 3 = Berry Map 
- 4 = Roaming Map 
- 5 = Trainer Map 

<Callout type="hint">
  When Normal Mode is not in this Setting, [WeatherOnLocationPreviewActive](#weatheronlocationpreviewactive) will have no effect.
</Callout>
