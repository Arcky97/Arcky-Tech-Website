## UseWeatherPreview

Use this Setting to enable or disable the Weather Preview Feature on the Region Map. This does require additional Setup but it's outside of this Plugin. More information you'll find in [Weather Preview Setup](../../plugin-setup/weather-preview)

```ruby
UseWeatherPreview = true
```

### Possible Values

- `true` = Weather Preview **<Gn>can be used</Gn>**.
- `false` = Weather Preview **<Rd>cannot be used</Rd>**.

<Callout type="hint">
  This Setting has no effect when the Weather Plugin is not installed.
</Callout>
