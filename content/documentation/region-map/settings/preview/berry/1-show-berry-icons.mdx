## ShowBerryIcons

Use this Setting to enable or disable the Berry Preview Feature on the Region Map.

```ruby
ShowBerryIcons = true 
```

### Possible Values

- `true` or `0` = Berry Preview **<Gn>can be used</Gn>**.
- `false` or `-1` = Berry Preview **<Rd>cannot be used</Rd>**.
- `Switch ID` = Berry Preview **<Or>can only be used</Or>** when the Game Switch is `ON`.

<Callout type="hint">
  There must be at least 1 planted Berry Event before the Berry Mode will be available.
</Callout>
