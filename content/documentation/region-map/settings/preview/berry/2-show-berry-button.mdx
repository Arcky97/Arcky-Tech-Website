## ShowBerryButton 

Use this Setting to set which button needs to be pressed to show the Berry Preview of planted Berries on the Region Map.

<SetVer
  v201="SHOW_BERRY_BUTTON = Input::USE"
  v211="ShowBerryButton = Input::USE"
/>

### Possible Values

- `Input::USE` = The Game Button.

<Callout type="hint">
  Be careful with which button you set this to as it can conflict with another button feature trigger. But you can set it to the same button as for [ShowLocationButton](location/#showlocationbutton)
</Callout>
