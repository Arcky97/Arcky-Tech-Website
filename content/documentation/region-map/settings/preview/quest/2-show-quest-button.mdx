## ShowQuestButton 

Use this Setting to set which button needs to be pressed to show the Quest Preview of a Quest on the Region Map.

```ruby 
ShowQuestButton = Input::USE
```

### Possible Values

- `Input::USE` = The Game Button.

<Callout type="hint">
  Be careful with which button you set this to as it can conflict with another button feature trigger. But you can set it to the same button as for [ShowLocationButton](location/#showlocationbutton)
</Callout>
