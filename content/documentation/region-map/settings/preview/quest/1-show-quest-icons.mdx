## ShowQuestIcons

Use this Setting to enable or disable the Quest Preview Feature on the Region Map.

```ruby
ShowQuestIcons = true 
```

### Possible Values

- `true` or `0` = Quest Preview **<Gn>can be used</Gn>**.
- `false` or `-1` = Quest Preview **<Rd>cannot be used</Rd>**.
- `Switch ID` = Quest Preview **<Or>can only be used</Or>** when the Game Switch is `ON`.

<Callout type="hint">
  There must be at least 1 Active Quest with a Map parameter defined before the Quest Mode will be available.
</Callout>
