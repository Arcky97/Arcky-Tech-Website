## ShowTrainerIcons

Use this Setting to enable or disable the Trainer Rematches Preview Feature on the Region Map.

```ruby
ShowTrainerIcons = true
```

### Possible Values

- `true` or `0` = Trainer Icons **<Gn>will be shown</Gn>**.
- `false` or `-1` = Trainer Icons **<Rd>will not be shown</Rd>**.
- `Switch ID` = Trainer Icons **<Or>will only be shown</Or>** when the Game Switch is `ON`.

<Callout type="hint">
  There must be at least 1 Trainer Rematch being active before the Trainer Mode will be available.
</Callout>
